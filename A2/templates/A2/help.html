Help

Although using the to-do list is not that complicated, here are a few tips:

  • You can safely press the 'Clear List' button; You will get a warning asking for your confirmation before the list gets deleted!
  • If you haven't done it already, try hovering over and pressing a task's name or date - you will get to see the description you gave to task!
  • If you want to close the description again, just press the task's name or date again.
  • As the task name and due date are required, it should not be possible to add a task without filling those fields first.
  • You can set a task as 'Important', which makes the task stand out more in the list by changing its colour.
